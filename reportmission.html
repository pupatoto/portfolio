Report Mission

This is a project I worked on during my internship at Gosoft I developed an iframe page using Next.js, integrated into the company’s HQ web system, to display all missions.
Each mission contains a varying number of checkpoints, referred to as "Tiers". The page shows how many times rewards have been claimed for each Tier. Users can also click
into a Tier to see daily reward claims, with the ability to filter by specific dates. The site aggregates reward claim data based on the selected date and allows exporting
the results as a CSV file.
