Report Mission

This is a project I worked on during my internship at Gosoft, Developed an internal iframe page using Next.js, integrated into the company’s HQ web system to display
missions and reward tracking. Each mission includes multiple "Tiers", showing the number of reward claims per tier.
Users can view daily claim data, filter by date, and export results to CSV.
